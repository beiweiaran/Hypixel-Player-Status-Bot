This is a Hypixel player status query bot running on Tencent QQ. It allows you to fetch player data such as:

  • Player level, karma, language, status
  • Bedwars stats (kills, beds, FKDR, etc.)
  • Skywars stats (KD, wins/losses, souls, coins, etc.)

This is a closed-source project. You can freely use the bot via QQ command interface.
